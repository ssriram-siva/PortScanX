# 🚀 PortScanX

<img src="https://github.com/ssriram-siva/PortScanX/blob/main/assets/logo.png" alt="PortScanX Banner" width="100%">

### Fast • Lightweight • Python TCP Port Scanner

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)
![License](https://img.shields.io/badge/License-MIT-green)
![Status](https://img.shields.io/badge/Status-Active-success)

A simple and efficient Python-based TCP port scanner built for learning networking, socket programming, and cybersecurity fundamentals.

</div>

---

## 📖 Features

- ⚡ Fast TCP port scanning
- 🔍 Detects open ports
- 🎯 Scan a single host
- 🧵 Lightweight and easy to use
- 💻 Cross-platform (Windows, Linux, macOS)
- 📚 Great for cybersecurity learning

## 🚀 Installation

```bash
git clone https://github.com/ssriram-siva/PortScanX.git

cd PortScanX

pip install -r requirements.txt
```

---

## ▶️ Usage

```bash
python PortScanX.py
```

Example:

```
Target IP : 127.0.0.1
Start Port: 1
End Port  : 1024
```

Output:

```
[+] Port 22 is OPEN
[+] Port 80 is OPEN
[+] Port 443 is OPEN
```
